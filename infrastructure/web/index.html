Features

강력한 기능

DeskOn은 원격 작업에 필요한 모든 기능을 제공합니다

종단간 암호화

모든 연결은 256-bit 암호화로 보호됩니다. 데이터가 안전하게 전송됩니다.

빠른 연결

최적화된 서버 인프라로 끊김 없는 원격 연결을 경험하세요.

크로스 플랫폼

Windows, macOS, Linux, Android, iOS 모든 기기에서 사용할 수 있습니다.

음성 통화

원격 연결 중 실시간 음성 통화로 원활하게 소통할 수 있습니다.

파일 전송

드래그 앤 드롭으로 손쉽게 파일을 주고받을 수 있습니다.

완전 무료

개인 사용자는 모든 기능을 무료로 이용할 수 있습니다.

How it works

3단계로 시작하세요

복잡한 설정 없이 바로 사용할 수 있습니다

1

앱 설치

위에서 운영체제에 맞는 버전을 다운로드하고 설치하세요.

2

ID 확인

앱을 실행하면 고유한 ID가 표시됩니다. 이 ID를 상대방에게 알려주세요.

3

연결하기

상대방의 ID를 입력하고 연결 버튼을 누르면 바로 원격 접속됩니다.
